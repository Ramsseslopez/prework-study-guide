  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element defines a section in a document that contains thematic content.
  • The meta element defines metadata such as a character set, description, keywords, author, and viewport.
  • The link element connects a external resource to the HTML document.
  • The header element appears at the top of the page and contains a navigation menu and/or introductory content.
  • h1-h6 elements represent the level of heading a given text block represents.
  • The paragraph element (p) represents a paragraph or block of text
  • The a element also known as a anchor creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link: Ex:More About Giant Pandas
